# Camera Controls Demo

This project demonstrates interactive camera controls in a 3D scene using React Three Fiber and Drei.
Users can move, rotate, and zoom the camera, and adjust movement steps and smoothness via a UI panel.

## Features

- 3D model viewer with environment lighting
- Camera controls (dolly, truck, rotate) with adjustable step sizes
- Section-based camera transitions
- UI controls powered by Leva

## Getting Started

Install dependencies:

```sh
yarn install
```

Start the development server:

```sh
yarn dev
```

## Tech Stack

- React
- Three.js
- @react-three/fiber
- @react-three/drei
- Leva
- Vite
